Including libraries with the Arduino IDE

First, find the library that you would like to include.

In this case I would like to down a DHT library for my DHT11 sensor.

Find a github repository or any download for a .zip file

In this case, since this is a github repository I just download the entire repository as a .zip file as shown below.

There is no need to extract the .zip file after download.

Simply open the Arduino IDE and navigate as shown in the image below:

After clicking on 'Add .zip library', simply select the .zip library through the file explorer.

After that is done, the added library should show up as shown below:

Now that the library has been included, simply include the relevant header files at the start of the program to be able to call upon their functions.
