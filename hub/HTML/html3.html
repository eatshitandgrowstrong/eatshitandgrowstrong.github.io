Unordered and Ordered Lists

Unordered lists

Unordered lists starts with a <ul> tag, with the items starting with a <li> tag. The listed items will be marked with black bullet points by default.

Here is an example:

The code would look like this:

<ul>

<li>Toronto</li>
<li>Montreal</li>
<li>Edmonton</li>
<li>Calgary</li>

</ul>

Here is what it would look like on the website

  • Toronto
  • Montreal
  • Edmonton
  • Calgary

Ordered lists

Ordered lists are like unordered lists, starting with a <ol> tag and like unordered lists, their items start with a <li> tag.

However, unlike unordered lists, instead of bullet points, the list items are automatically marked with numbers by default.

Here is another example but of an ordered list:

  1. Toronto
  2. Montreal
  3. Edmonton
  4. Calgary

Ordered lists makes creating a long list of items much easier with automatic numbering instead of having to manual number them.
